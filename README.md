## Hi there

I use Fedora 43 (i3 spin) with LUKS encryption daily. My Linux journey started with Ubuntu in a dual-boot configuration alongside Windows, and I've since explored various distributions — Arch Linux (minimal install for my homelab), Debian (my go-to for VPS servers), and several desktop environments. Ultimately, I prefer the efficiency of a window manager and appreciate a minimal memory footprint. I favor the keyboard over the mouse — I use terminal-based code editors, having tried Neovim but now preferring Helix.

I'm passionate about systems programming and building high-performance web services that scale efficiently with minimal resources. My interests lie in distributed, decentralized, and P2P systems.

I have several years of experience in Python and am currently learning Rust. I also leverage AI tools for research and learning.

I care deeply about internet privacy and digital sovereignty. I like various cryptocurrencies, especially Monero. I believe in the power of free and open-source software to protect user freedom.
